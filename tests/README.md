# Tests Directory

This directory contains all test files for the cricket object identifier project.

## Running Tests

To run all tests:
```bash
pytest tests/
```

To run tests with coverage:
```bash
pytest --cov=cricket_object_identifier tests/
```

## Test Structure

- Test files should be named `test_*.py`
- Test functions should be named `test_*`
- Follow the existing test patterns in the repository
